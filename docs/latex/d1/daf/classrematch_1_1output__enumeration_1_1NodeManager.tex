\doxysubsection{Detailed Description}
The Node Manager class stores the pointers to all allocated \mbox{\hyperlink{classrematch_1_1output__enumeration_1_1ECSNode}{ECSNode}}\textquotesingle{}s. 

When an \mbox{\hyperlink{classrematch_1_1output__enumeration_1_1ECSNode}{ECSNode}} is no longer used, i.\+e, when the amount of references to it has become 0, that memory is available to be recycled.
