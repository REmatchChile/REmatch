\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classrematch_1_1output__enumeration_1_1Mapping}{Mapping}} is also known as an s-\/tuple, asigns a span to all it\textquotesingle{}s variables. 

This is an internal mapping, therefore the spans are assigned to the variable id\textquotesingle{}s, to get the original variable names the Variable\+Catalog class should be used.
